> `Math`、`English`、`Graphic Demos`

**[数学工具](https://www.desmos.com/)**
> `计算器`、`矩阵`、`图形`
>
> 免费的数学软件工具(在线版)，助你不断探索数学的奥妙。

**[沉浸式数学](http://immersivemath.com/)**
> `线性代数`、`大量图片`、`交互案例`
>
> 就是一本教你线性代数知识的数学小本子。用它学习线性代数的基础，还是很靠谱的。

**[几何画板](https://www.geogebra.org/)**
> `教学工具`、`函数画图`
>
> 免费的课堂活动数字工具, 用于绘图计算, 几何作图, 白板协作等等。

**[元数](https://metanumbers.com/)**
> `数字`、`数学常识`、`数字算法`
>
> 就专注于一个东西：正整数。1、2、100……这些数字都有哪些不为人知的秘密？本站对它们一网打尽！

**[物理仿真](https://www.myphysicslab.com/)**
> `Physics`、`English`、`Animation`
>
> 动画化、可交互化，妥妥的一个力学线上实验室。

**[探索物理](https://landgreen.github.io/physics/index.html)**
> `Physics`、`English`、`物理笔记`
>
> 一个搭建在 Github 上的带有交互模拟器的站点，涉及力学、光学、磁学、电学等等。因为基于 Github 嘛，最好带上你的梯子。

**[物理超级课本](https://physics.info/)**
> `Physics`、`English`、`物理课本`
>
> 囊括了声光热力电，它可不仅仅是一本物理教科书，更是一份作者探索物理奥秘的热情结晶。

**[傅里叶变换](https://www.jezzamon.com/fourier/)**
> `傅里叶`、`数学`、`信息论`
>
> 无论你在开始学习数学、电子、计算机图形学，亦或其他的理学工学，都不应当错过该教程。这是理论基础，是要掌握的！

**[英语词汇](https://www.vocabulary.com/)**
> `English`、`Vocabulary`、`Dictionary`

> 一个专门记单词的地方，对单词的解析特别生动详细，英文学习者建议多逛逛。

**[趣词](https://www.quword.com/root/)**
